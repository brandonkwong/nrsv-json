## Build Args

When editing JSON [structure](src/lib/structure.js), the `--indent` arg can be used to adjust spacing (default is `4`) for desired readability.

```
yarn build --indent 2
```

[nrsv.json](nrsv.json) is stringified for distribution.

```
yarn build --dist
```
